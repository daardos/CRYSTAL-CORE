🌌 Crystal Archive Pro
Crystal Archive Pro — это иммерсивный текстовый редактор нового поколения, сочетающий в себе эстетику Glassmorphism и мощный функционал настольных текстовых процессоров. Это не просто блокнот, а цифровая среда для творчества, парящая в «туманной бездне».
✨ Основные особенности
🔮 Визуальное совершенство
Frosted Glass Engine: уникальный эффект запотевшего стекла с динамическим размытием фона (фильтр Backdrop).
Фон Deep Abyss: интерактивный фон с математической симуляцией частиц, реагирующих на движение.
Golden Accent: премиальная цветовая палитра «Полуночное золото» для максимальной концентрации и комфорта для глаз.
📝 Профессиональный редактор
WYSIWYG Word-Logic: полноценное форматирование текста (жирный шрифт, курсив, подчеркивание, выделение золотым).
Поддержка мультимедиа: вставка изображений и файлов простым перетаскиванием (Drag-and-Drop).
Умная иерархия: двухуровневая архитектура «Тома и Листы» для управления большими объемами информации.
🛠 Технологический стек
Instant Ghost-Save: система мгновенного автосохранения в LocalStorage. Ваши данные не пропадут даже при закрытии браузера.
Глобальный поиск: мгновенный поиск по всему архиву (включая содержимое всех страниц).
Печать и PDF: интеллектуальный экспорт. Сохраняет цвета в PDF и автоматически адаптирует верстку для черно-белой печати на принтере.
🚀 Быстрый старт
Проект представляет собой одностраничное приложение (SPA) и не требует установки тяжелых зависимостей.
Скачайте файл index.html.
Откройте его в любом современном браузере (Chrome, Edge, Safari, Firefox).
Начните творить.
🏗 Технические подробности
Проект реализован на чистом стеке без использования тяжелых фреймворков, что обеспечивает молниеносную скорость работы:
HTML5 ContentEditable API — для гибкого редактирования текста.
CSS3 Grid & Flexbox — для адаптивной и сложной сетки интерфейса.
Canvas API — для высокопроизводительной анимации фона.
HTML2PDF.js — для генерации векторных документов на лету.
